Support

Everything you need to try, buy, install, and get the most out of Cognate blocks on your Darkglass Anagram.

Quick start

1

Try

Open the Anagram Marketplace Cognate Audio vendor page. Every block has a free trial — load it on your Anagram and play.

2

Buy

When you're ready, buy directly from the Marketplace listing. Cognate Metro is free; the rest are one-off purchases — no subscriptions, yours forever.

3

Install

Check that your Anagram is registered on the MyPedal website. Try or Buy a block and click Add to Library. Log in to Darkglass Suite with Anagram connected - you should be prompted to sync your new plugin. Drop it into any preset.

FAQs

Where do I buy Cognate blocks?

All our blocks are available exclusively on the Anagram Marketplace.

Do I need an Anagram to use Cognate blocks?

Yes. Cognate blocks are designed specifically for the Darkglass Anagram effects processor. They aren't available as VST/AU/AAX plugins currently.

Do these work on guitar?

Bass-first means bass isn't an afterthought — but it doesn't mean bass-only. Most of our blocks sound great on guitar too: Bitcrush, Ringmod, Hologram, Pultrick and Kinetic all work happily across the full string-instrument range. Cognate Metro is universal by design. Cognate Chord is the one block that's specifically a bass-practice tool, but you'll find guitarists using it too. Try anything you like with the free Marketplace trial.

Which Anagram KosmOS version do I need?

Our blocks should run on most recent versions, but we always recommend using the latest KosmOS release. You can update via the Darkglass Suite.

Can I try before I buy?

Yes. Every block has a free trial on the Anagram Marketplace — load it on your Anagram and try it for yourself. In trial mode, blocks will become silent for 5 seconds every 30 seconds.

Will my preset still work after a block update?

Yes — preset compatibility is a hard guarantee. We don't change parameter ranges, defaults or behaviour in ways that would invalidate a saved preset. If we ever need to make a breaking change, we'd ship it as a new block rather than overwrite the existing one.

How do refunds work?

Purchases and refunds are handled by the Anagram Marketplace. See their support pages for the current refund policy.

I bought a block but I don't see it on my Anagram

Check that your Anagram is registered on the MyPedal website. On the Marketplace, Log In with the same email address and check that the block appears in your Library. Then start Darkglass Suite with Anagram connected, and log in with the same email address. You should be prompted to install any plugins you have purchesed, and you can manage your plugins in the Assets tab. Check for the block in the category matching the one on the Anagram Marketplace. If it's still missing, get in touch with Darkglass Support.

Can you make a block for X?

We love hearing block ideas. Drop us a feature request via the contact form — even if we don't end up making it ourselves, it helps us understand what bassists actually want.

Troubleshooting

A block I purchased doesn't appear on Anagram

Check that you are signed into Anagram Marketplace and Darkglass Suite with the same email address. Once signed in, Darkglass Suite should prompt you to install your block.

Audio cuts out when a Cognate block is on

In trial mode, any block will become silent for 5 seconds every 30 seconds. This will affect the entire signal chain, regardless of block Bypass settings.

I see a red DSP Overload warning

Anagram can support many blocks running simultaneously but some blocks consume more CPU resources than others. NAM loader blocks in particular can use more CPU resources depending on how they were captured. As a result, it's not possible to guarantee that all Anagram blocks can run simultaneously. If you see a red DSP Overload warning, you should try to simplify the signal chain by removing or bypassing some blocks. We work hard to optimise and streamline our blocks to use as little CPU as possible. We publish our blocks' CPU usage on this website.

A parameter doesn't behave the way the manual describes

Make sure you're on the latest version of the block. Sync your Anagram to Darkglass Suite to install any updated versions of the block, and update KosmOS. If that doesn't work, tell us and we'll dig in.

How do I reset a block to its defaults?

Open the block from your signal chain. Tap the three dots icon and choose "Restore factory default state".
